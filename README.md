# Sistema de Vendas e Inventario - Loja Queiroz Lubrificantes

## Requesitos do Projeto

- Sistema para cadastrar e gerenciar produtos
  - Cadastrar produtos
  - Listar produtos
  - Alterar produtos
  - Excluir produtos
  - Modificar a quantidade em estoque

- Sistema para efetuar vendas
  - Cadastrar vendas
  - Listar vendas
